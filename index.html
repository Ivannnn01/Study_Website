📋 Exam CSV Format Guide
Column A
Exam Name
Any text, e.g. H2 Chemistry Paper 1
→
Column B
Exam Date
18/06/2026   2026-06-18   18 June 2026
→
Column C — optional
Exam Type
e.g. A-Levels, Prelims, Mid-Year
✓ Header row is optional — if Column A is “name” / “exam” and Column B is “date”, that row is auto-skipped.
✓ Date formats accepted: DD/MM/YYYY, YYYY-MM-DD, 18 June 2026, June 18 2026 and more.
✓ Column C (Exam Type) is completely optional — rows with only 2 columns import fine.
! Rows with unreadable dates are skipped and reported in a toast notification.
Example (with header + exam type):
Exam Name,Exam Date,Exam Type
H2 Chemistry P1,18/06/2026,Prelims
H2 Econs P2,19/06/2026,Prelims
H2 Math P1,2026-11-03,A-Levels
Example (minimal, no header):
H2 Chemistry P1,18/06/2026
H2 Econs P2,19 June 2026
